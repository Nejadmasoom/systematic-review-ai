# 📋 SysReview AI — دستیار رویوی سیستماتیک

ابزار رایگان و متن‌باز برای کمک به محققان نوروساینس، روانشناسی و پزشکی در:
- Parse کردن سرچ استراتژی Scopus / Web of Science / PubMed / Embase با AI
- **وارد کردن مستقیم فایل RIS / BibTeX / CSV** (خروجی Zotero، EndNote، PubMed، Scopus) — بدون تایپ دستی
- ساخت معیارهای ورود/خروج بر اساس چارچوب استاندارد **PICO/PECO**
- **غربال‌گری دسته‌ای** ده‌ها مقاله هم‌زمان با AI، با رعایت محدودیت نرخ APIهای رایگان
- تشخیص داپلیکیت با DOI دقیق **و** شباهت فازی عنوان (برای مقالاتی که DOI ندارند یا فرمتشان کمی فرق دارد)
- نمودار **PRISMA واقعی (SVG قابل دانلود)** برای درج مستقیم در مقاله
- خروجی CSV و RIS (برای بازگشت به Zotero/EndNote)

تمام داده‌ها (کلید API، مقالات، استراتژی) فقط در مرورگر خودتان (`localStorage`) ذخیره می‌شوند — هیچ سروری در میان نیست.

## 🔑 گرفتن کلید API (رایگان)

### Groq (پیشنهاد می‌شود — رایگان و بسیار سریع)
1. به [console.groq.com](https://console.groq.com) بروید
2. ثبت‌نام کنید (با Google امکان‌پذیر است)
3. `API Keys → Create API Key`
4. کلید را کپی کرده و در تب «تنظیمات API» این ابزار وارد کنید
5. مدل پیشنهادی: **Llama 3.3 70B** (دارای نشان ⭐)

### Google AI Studio / Gemini (رایگان با محدودیت روزانه)
1. به [aistudio.google.com/apikey](https://aistudio.google.com/apikey) بروید
2. با اکانت گوگل وارد شوید
3. `Create API Key`
4. مدل پیشنهادی: **Gemini 2.0 Flash**

### OpenAI / ChatGPT (پولی، اما دقیق‌ترین)
1. به [platform.openai.com/api-keys](https://platform.openai.com/api-keys) بروید
2. کلید بسازید (نیاز به شارژ حساب دارد)
3. می‌توانید مدل‌های Claude و Gemini را هم از طریق سرویس‌های سازگار با OpenAI API (مثل OpenRouter) با تغییر **Base URL** استفاده کنید:
   - برای OpenRouter: `https://openrouter.ai/api/v1`
4. مدل پیشنهادی: **GPT-4o** یا **Claude Sonnet 4.6** (در صورت استفاده از OpenRouter)

> 💡 **نکته:** کلید API هرگز به هیچ سروری جز سرور همان provider ارسال نمی‌شود. درخواست‌ها مستقیماً از مرورگر شما به OpenAI/Google/Groq می‌روند.

## 🧭 راهنمای استفاده

### ۱. تب «استراتژی جستجو»
سرچ استراتژی که از Scopus، WoS یا PubMed دارید را در باکس سمت چپ paste کنید و روی «Parse با AI» بزنید. هوش مصنوعی کلیدواژه‌ها، MeSH termها، و ساختار Boolean را استخراج می‌کند.

سمت راست، معیارهای ورود را بر اساس چارچوب استاندارد **PICO/PECO** پر کنید (جمعیت، مداخله، مقایسه، پیامد). این چارچوب مستقیماً دقت غربال‌گری AI را بالا می‌برد چون مدل می‌داند دقیقاً دنبال چه چیزی بگردد. معیارهای سفارشی اضافه هم در بخش «پیشرفته» قابل افزودن هستند.

### ۲. تب «غربال‌گری AI»
**برای یک مقاله:** عنوان و چکیده را وارد کرده و «غربال با AI» را بزنید.

**برای چند مقاله هم‌زمان (پیشنهادی برای آپدیت دوره‌ای):**
1. از Zotero/EndNote/PubMed/Scopus خروجی `.ris` یا `.bib` بگیرید
2. فایل را در بخش «ورود و غربال‌گری دسته‌ای» آپلود یا متن آن را paste کنید — فرمت خودکار تشخیص داده می‌شود
3. روی «شروع غربال‌گری دسته‌ای» بزنید — AI یکی‌یکی مقالات غربال‌نشده را بررسی می‌کند (با فاصله زمانی مناسب برای پلن‌های رایگان Groq/Gemini تا با خطای rate-limit مواجه نشوید)

در همه حالت‌ها، اگر اطلاعات کافی برای قضاوت قطعی نباشد، AI وضعیت «در بررسی» برمی‌گرداند و حدس نمی‌زند.

### ۳. تب «مقالات»
تمام مقالات با وضعیت (شامل/خارج/در بررسی)، دلیل AI، و امکان ویرایش دستی. داپلیکیت‌ها هم با DOI دقیق و هم با شباهت بالای عنوان (برای مقالات بدون DOI) خودکار رد می‌شوند.

### ۴. تب «آپدیت دوره‌ای»
تاریخ آخرین جستجو را ثبت کنید. ابزار لینک‌های آماده برای Google Scholar، PubMed، Semantic Scholar، Europe PMC، CORE و OpenAlex می‌سازد — بر اساس کلیدواژه‌های استخراج‌شده. یک چک‌لیست هم برای پیگیری مراحل آپدیت دارید.

### ۵. تب «خروجی»
نمودار **PRISMA واقعی** (SVG قابل دانلود و درج مستقیم در مقاله) + خروجی CSV (سازگار با Excel، Rayyan) + خروجی **RIS** (برای بازگشت به Zotero/EndNote) + متن آماده برای بخش Methods مقاله.

## ⚠️ محدودیت‌های مهم

- این ابزار **جایگزین قضاوت انسانی نیست** — همیشه نتایج AI را بازبینی کنید، به‌خصوص مقالات «در بررسی» و موارد نزدیک به خط مرزی
- دقت AI به کیفیت معیارهای ورود/خروجی که وارد می‌کنید بستگی دارد — هرچه دقیق‌تر، نتیجه بهتر
- مدل‌های رایگان (Groq, Gemini free tier) محدودیت نرخ درخواست دارند؛ برای حجم بالا صبر بین درخواست‌ها لازم است
- داده‌ها فقط در مرورگر فعلی شما ذخیره می‌شوند — برای انتقال بین دستگاه‌ها از خروجی CSV استفاده کنید
